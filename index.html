JSON y JavaScript

Introducción

En este sitio web, hablaremos de los fichero de tipo JSON. El acrónimo de JSON (JavaScript Object Notation), a grandes rasgos, es un formato de texto sencillo y ligero que nos permite intercambiar datos de manera más eficaz. Este tipo de ficheros, de manera vulgar se considera que son una array, pero eso es una blasfemia. Los ficheros JSON son una representación sencilla de lo que entendemos cómo un objeto. Además, no pesa demasiado y carece de claves primarias y foráneas.

Tipos de datos

Tal y cómo hemos dicho en el anterior apartado. Un fichero JSON es una representación sencilla de un objeto. Está estructurado en propiedades y valores. Las propiedades pueden tener cómo valor cadenas de caracteres, números, colección de datos, colección de objetos y valores booleanos.

  1. string: Este tipo de variable acepta cadenas de caracteres.
  2. numérica: Acepta todo tipo de números ( Integer, Double, Float, etc. ).
  3. booleanos: Acepta dos valores true o false.
  4. Colección de datos:
    • Acepta una colección de valores.
    • Acepta una colección de objetos con sus propiedades.
